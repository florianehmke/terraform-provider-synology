# Core: Task Run (Resource)

Runs an existing DSM Task Scheduler entry once during resource creation or replacement.

This resource is intended for one-shot bootstrap flows. Re-run it with `terraform apply -replace=...`
when you need to invoke the same DSM task again.

## Schema

### Required

- `task_name` (String) Exact DSM task name to run.

### Read-Only

- `id` (String) Stable resource ID, equal to `task_name`.
- `last_run_at` (String) UTC RFC3339 timestamp recorded when the provider last invoked the task.
- `task_id` (Number) Resolved DSM task ID for `task_name`.
