# Core: Certificate Service Binding (Resource)

Binds a DSM certificate to a DSM service entry.

This resource is intended for services that DSM already knows about, such as reverse-proxy
hosts. On destroy, the service is rebound to the DSM default certificate.



<!-- schema generated by tfplugindocs -->
## Schema

### Required

- `certificate_id` (String) Target DSM certificate ID.
- `display_name` (String) DSM service display name, such as a reverse-proxy hostname.

### Optional

- `subscriber` (String) DSM service subscriber. Default: `ReverseProxy`.

### Read-Only

- `id` (String) Binding identifier in the form `subscriber/display_name`.
