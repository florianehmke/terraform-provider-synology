# Filestation: Path (Data Source)

Looks up a File Station path on the NAS.

When `wait_for_exists` is true, the data source waits until the path appears or the timeout
expires. When it is false and the path does not exist, the data source returns `exists = false`
instead of failing.



<!-- schema generated by tfplugindocs -->
## Schema

### Required

- `path` (String) NAS path rooted at a shared folder, such as `/docker/certbot/current/cert.pem`.

### Optional

- `wait_for_exists` (Boolean) Wait for the path to exist before returning. Default: `false`.
- `wait_timeout_seconds` (Number) Maximum time to wait when `wait_for_exists` is true. Default: `300`.

### Read-Only

- `exists` (Boolean) Whether the path currently exists.
- `id` (String) Stable identifier for the lookup, equal to `path`.
- `is_dir` (Boolean) Whether the path is a directory.
- `name` (String) Basename reported by File Station.
- `real_path` (String) Resolved real path reported by File Station.
